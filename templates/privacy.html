Privacy Policy for Flags Bot

Last Updated: October 21, 2025

This Privacy Policy outlines how Flags Bot ("the Bot") collects, uses, and shares information when you use it within a Discord server.

1. Information We Collect

To provide its features, the Bot collects and stores the following information:

  • User Provided Information:
    • Your Discord User ID.
    • Your birthday (month and day only), if you choose to set it with the /birthday set command.
    • Your answers to introduction questions if you participate in the AI Introduction feature.
    • Banner images you upload via the dashboard.
  • Automatically Collected Information:
    • The Discord Server (Guild) ID where the Bot is used.
    • Game scores, levels, and XP from in-bot games like the flag quiz.
    • Spam offense counts for moderation purposes.
    • Nickname history when your nickname is changed.
    • Command usage logs (user ID, guild ID, command name, timestamp).
    • OAuth2 tokens (access and refresh) if you authorize via the website.
  • Temporarily Processed Information:
    • The content of your messages when you directly interact with the bot (e.g., by mentioning it or replying to it) is processed by third-party AI services to generate a response. This content is not stored long-term by the Bot.
    • The content of recent, highly-reacted-to messages within a channel is processed by third-party AI services to generate weekly highlights when the /highlights command is used. This data is processed in-memory and is not stored by the Bot.
    • hCaptcha responses via the website for verification.

2. How We Use Your Information

The data collected is used exclusively to operate the Bot's features, including but not limited to:

  • Providing game and leaderboard functionality.
  • Announcing user birthdays in a configured channel.
  • Moderating spam and inappropriate nicknames.
  • Generating AI-powered chatbot responses and user introductions.
  • Creating weekly server highlights.
  • Enabling website dashboard functionality and server management.
  • Facilitating the community recovery feature (`?pull`, `?recover`) using stored OAuth2 tokens.
  • Displaying welcome banners using uploaded images.
  • Logging interactions for analytics displayed on the owner dashboard.
  • Verifying human users on the website via hCaptcha.

3. Sharing with Third Parties

To provide AI-powered features, message content is shared with the following third-party services:

  • Groq: For generating chatbot responses, user introductions, and analyzing weekly highlights.
  • Mistral AI: As a fallback for generating chatbot responses.
  • Google Gemini: As a fallback for generating chatbot responses.

Additionally, for website verification:

  • hCaptcha: User interaction data is shared with hCaptcha when you complete the verification step on our website.

Uploaded banner images are stored via Discord's CDN after being posted to a private channel. This data is sent to these services solely for the purpose of generating a response or analysis for the Bot's features or verifying website access. We do not sell or share your personal data with any other third parties for marketing or unrelated purposes.

4. Your Rights and Data Control

You have control over some of your data:

  • You can remove your birthday information at any time by using the /birthday remove command.
  • You can deauthorize the application via Discord's User Settings -> Authorized Apps, which invalidates the stored OAuth2 tokens.
  • To request the removal of all other data associated with your User ID (scores, logs, etc.), please contact the bot operator. Note that removing necessary data may impact your ability to use certain bot features.
